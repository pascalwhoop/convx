# convx explore — Keyboard Navigation

The explore TUI has three focusable areas: the search bar, the session list, and the markdown preview. Use **Tab** and **Shift+Tab** to move focus between them.

## Search Bar

| Key | Action |
|-----|--------|
| Type | Fuzzy-filter sessions by title (instant) |
| **Enter** | Run full-text search across conversation content |
| **Escape** | Clear search, restore full session list |
| **Tab** | Move focus to session list |

## Session List

| Key | Action |
|-----|--------|
| **↑** / **↓** or **k** / **j** | Move highlight up/down |
| **g** / **G** | Jump to top / bottom |
| **Enter** | Select highlighted session (loads preview) |
| **Tab** | Move focus to preview pane |
| **Shift+Tab** | Move focus back to search bar |

## Preview Pane

| Key | Action |
|-----|--------|
| **↑** / **↓** | Scroll up/down |
| **Page Up** / **Page Down** | Scroll by page |
| **Home** / **End** | Jump to top/bottom |
| **Tab** | Move focus to search bar |
| **Shift+Tab** | Move focus back to session list |

## Global

| Key | Action |
|-----|--------|
| **q** | Quit |
